included(base)

This method adds the localized_model_name and the human_attribute_name to the ActiveRecord::Base class.

localized_model_name looks for the localized model name in the language file and human_attribute_name looks for the localized attribute names. localized_model_name returns nil if no matching entry could be found. human_attribute_name calls the super method if no entry matches and therefore falls back to Rails default behavior.
